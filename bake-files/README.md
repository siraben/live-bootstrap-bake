# bake files

This directory mirrors the `.bake` files introduced by the patch stack at their
intended upstream paths.

The files under `seed/stage0-posix/` cover the stage0 handoff from kaem to
`bake`. The files under `steps/` cover the tracked kaem-era
`live-bootstrap` `pass1` package wrappers through `bash-2.05b`.

The patches remain authoritative for applying the changes; these copies are
included so the recipes can be reviewed directly.
